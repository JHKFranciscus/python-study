## 4. 문제의 원인

[내가 혼자 원인을 알아냈다고 쓰지 않는다.]

사실에 맞는 구분:

- 내가 직접 확인한 것:
  - 프로그램 실행 중 오류 메시지가 나타난 것
  - 전체 보기 결과가 정상적이지 않았던 것

- 설명을 받아 알게 된 것:
  - `scores = new_score`는 리스트에 점수를 추가하는 것이 아니라
    `scores` 자체를 정수로 바꾼다는 점
  - `scores.append(scores)`는 리스트 안에 자기 자신을 넣는다는 점
  - 새 점수를 추가하려면 `scores.append(new_score)`를 사용해야 한다는 점

[위 내용을 본인 문장으로 3~6문장 작성한다.]

## 6. 수정한 방법

[수정 전 코드와 수정 후 코드를 나란히 적는다.]

수정 전:

```python
scores = new_score
```

또는

```python
scores.append(scores)
```

수정 후:

```python
scores.append(new_score)
```

[각 코드가 무엇을 하는지 2~4문장으로 설명한다.]

추가로 통계 계산에서는 `total`, `highest_score`, `smallest_score`를
반복문 안에서 매번 초기화하지 않고 반복문 전에 준비했다.

[실행 결과를 글로 적거나 터미널 화면을 캡처해 첨부한다.]

확인한 결과:

- 빈 리스트일 때 안내 문장이 출력됨
