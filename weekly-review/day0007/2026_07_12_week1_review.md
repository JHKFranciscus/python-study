## Git

8. git status, git add, git commit, git push의 역할은 각각 무엇인가?

git status: 현재 git에 저장되어 있거나 저장되지 않은 파일을 보여준다.
git add: 현재 새로 변화한 파일이나 폴더를 스테이징 영역에 올려두는 것
git commit: git add로 올려둔 변화된 자료를 실제로 git에 저장하는 것
git push: git에 commit된 자료를 GitHub에 올리는 것

[처음 답에서 부족하거나 틀린 부분]
git status: 현재 파일 상태를 확인
git add: 현재 변경 내용을 다음 commit에 포함할 대상(스테이징 영역)으로 올려둔다.
git commit: git add로 올려둔 변경 내용을 하나의 기록으로 저장(로컬 Git 저장소에 기록)
git push: 로컬 commit을 GitHub 원격 저장소로 전송

### 5. 오류가 발생한 부분과 도움받은 내용

- `scores = new_score`라고 작성해 `scores`가 리스트가 아니라 정수로 바뀌었다.
- 전체 보기에서 `len(scores)`를 실행하자
  `TypeError: object of type 'int' has no len()` 오류가 발생했다.
- 나는 오류 원인을 바로 찾지 못했고, 설명을 받은 뒤
  `scores.append(new_score)`로 수정해야 한다는 것을 확인했다.
- 수정 과정에서 다시 `scores.append(scores)`라고 작성해
  리스트 안에 자기 자신을 넣었다.
- 출력이 `[[...], [...]]` 형태로 나온 뒤,
  다시 설명을 받고 `scores.append(new_score)`로 수정했다.

### 6. 내가 직접 확인한 실행 결과

- 전체 보기에서 `len(scores)`를 실행하자
  `TypeError: object of type 'int' has no len()` 오류가 발생한 것을 직접 확인했다.

- 오류 원인은 바로 알아내지 못했고, 설명을 받은 뒤
  `scores = new_score` 때문에 `scores`가 정수로 바뀌었다는 것을 확인했다.

- 설명에 따라 `scores.append(new_score)`로 수정한 뒤
  전체 보기에서 `[80, 95]`가 출력되는 것을 직접 확인했다.
